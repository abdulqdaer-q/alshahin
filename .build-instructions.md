# Quick Build Instructions

This file provides a quick reference for building the app. For comprehensive instructions, see [BUILDING.md](BUILDING.md).

## First Time Setup

```bash
# Install dependencies
npm install

# Build TypeScript
npm run build
```

## Development Build

```bash
# Build and run (unsigned)
./build_and_package.sh --skip-signing --skip-notarize
```

## Production Build

```bash
# Set up signing (required once per session)
export IDENTITY_APP="Developer ID Application: Your Name (TEAM_ID)"
export IDENTITY_INSTALLER="Developer ID Installer: Your Name (TEAM_ID)"
export APPLE_ID="your-apple-id@example.com"
export APPLE_ID_PASSWORD="xxxx-xxxx-xxxx-xxxx"
export APPLE_TEAM_ID="XXXXXXXXXX"

# Build signed and notarized app
./build_and_package.sh
```

## Quick Commands

```bash
# Apple Silicon only
./build_and_package.sh --arm64-only

# Universal binary
./build_and_package.sh --universal

# Clean build
./build_and_package.sh --clean

# Skip tests
./build_and_package.sh --skip-tests
```

## Files Created

- `electron-builder.yml` - Build configuration
- `build_and_package.sh` - Automated build script
- `build/entitlements.mac.plist` - macOS entitlements
- `build/notarize.js` - Notarization script
- `tests/build/buildConfig.test.ts` - Build validation tests
- `BUILDING.md` - Comprehensive build guide

## Architecture

- **arm64**: Apple Silicon (M1/M2/M3/M4)
- **x64**: Intel Macs
- **universal**: Both (larger file size)

## Output

Build artifacts are in `dist/`:
- `*.app` - Application bundle
- `*.dmg` - Disk image installer
- `*.pkg` - Package installer
- `*.zip` - Compressed app

For more details, see [BUILDING.md](BUILDING.md).
